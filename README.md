# spin-vite-ts

Minimal-ish working example of using vite, itty-router, and spin framework.


### Prerequisites:

- spin
- bun
- node


## Quickstart

```
git clone https://github.com/seemueller-io/spin-vite-ts.git
bun spin:dev
open http://localhost:3000
```
